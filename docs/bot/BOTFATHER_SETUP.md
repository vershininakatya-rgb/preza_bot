# Инструкция по настройке бота в BotFather

Пошаговое руководство по созданию и настройке Telegram-бота для подготовки презентаций в [@BotFather](https://t.me/BotFather).

---

## Шаг 1. Создание бота

1. Откройте Telegram и найдите **@BotFather** (официальный бот Telegram для управления ботами).
2. Отправьте команду `/newbot`.
3. BotFather спросит **имя бота** (отображается в профиле):
   - Пример: `Preza Bot` или `Бот для презентаций`
   - Имя может быть на любом языке.
4. BotFather спросит **username бота** (должен заканчиваться на `bot`):
   - Пример: `preza_presentation_bot` или `my_presentation_bot`
   - Username должен быть уникальным и содержать только латиницу, цифры и подчёркивания.
5. После успешного создания BotFather выдаст **токен** вида:
   ```
   7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
6. **Сохраните токен** — он понадобится для файла `.env`.

---

## Шаг 2. Добавление токена в проект

1. Скопируйте токен из BotFather.
2. Откройте файл `.env` в корне проекта (если его нет — скопируйте из `.env.example`).
3. Вставьте токен:
   ```
   BOT_TOKEN=7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
4. **Важно:** не публикуйте токен в репозиторий. Файл `.env` должен быть в `.gitignore`.

---

## Шаг 3. Настройка команд бота (BotFather)

Чтобы у пользователей отображалось меню команд при нажатии на `/`, настройте их в BotFather:

1. Отправьте BotFather команду `/setcommands`.
2. Выберите вашего бота из списка.
3. Отправьте список команд в формате:
   ```
   start - Начать работу с ботом
   help - Справка и доступные команды
   ```

BotFather примет список, и команды появятся в меню бота.

---

## Шаг 4. Описание бота (рекомендуется)

### Краткое описание (Short description)

Показывается в результатах поиска и в профиле бота. Лимит: 120 символов.

1. Отправьте BotFather команду `/setdescription`.
2. Выберите бота.
3. Введите описание:
   ```
   Для деливери-менеджеров и скрам-мастеров: аналитика, вопросы для интервью, презентации для C-level.
   ```

### О боте (About)

Показывается в профиле бота при нажатии на «О боте». Лимит: 512 символов.

1. Отправьте BotFather команду `/setabouttext`.
2. Выберите бота.
3. Введите текст:
   ```
   Для кого: деливери-менеджеры, скрам-мастеры, change-менеджеры, агенты изменений (компании 300+ человек).

   С чем помогает:
   • Помощь с аналитикой — структурировать анализ проблемы, план исследования, метрики, дерево анализа
   • Вопросы для интервью — шаблоны вопросов по ролям (C-level, руководители, тимлиды, аналитики)
   • Подготовка презентации — от черновика до сборки в корпоративном шаблоне (PPTX)
   • Полный цикл — анализ + вопросы + презентация для встреч с руководством
   ```

## Шаг 5. Дополнительные настройки (по желанию)

| Команда BotFather | Назначение |
|-------------------|------------|
| `/setname` | Изменить имя бота |
| `/setdescription` | Краткое описание (до 120 символов) |
| `/setabouttext` | Текст «О боте» (до 512 символов) |
| `/setuserpic` | Загрузить аватар бота |
| `/deletebot` | Удалить бота |

---

## Шаг 6. Проверка работы

1. Убедитесь, что в `.env` указан корректный `BOT_TOKEN`.
2. Запустите бота:
   ```bash
   python run.py
   ```
3. Найдите бота в Telegram по username (например, `@preza_presentation_bot`).
4. Отправьте команду `/start` — бот должен ответить приветствием.

## Частые проблемы

### Бот не отвечает
- Проверьте, что токен в `.env` скопирован полностью, без пробелов.
- Убедитесь, что бот запущен (`python run.py`).
- Проверьте интернет-соединение.

### «Unauthorized» или ошибка 401
- Токен неверный или устарел. Создайте новый токен в BotFather командой `/token` → выберите бота → «Revoke current token» и получите новый.

### Команды не отображаются в меню
- Убедитесь, что вы выполнили `/setcommands` в BotFather для вашего бота.
- Перезапустите чат с ботом (закройте и откройте заново).
