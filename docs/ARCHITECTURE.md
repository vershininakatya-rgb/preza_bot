# Архитектура приложения

## Общая структура

```
bot/
├── main.py              # Точка входа приложения
├── config/              # Конфигурация
│   └── settings.py
├── handlers/            # Обработчики событий
│   ├── commands.py
│   └── messages.py
└── utils/               # Утилиты
    ├── logger.py
    └── helpers.py
```

## Компоненты системы

### 1. Точка входа (main.py)
- Инициализация бота
- Регистрация обработчиков
- Запуск polling

### 2. Конфигурация (config/)
- Загрузка переменных окружения
- Настройки приложения
- Управление секретами

### 3. Обработчики (handlers/)
- **commands.py**: Обработка команд бота
- **messages.py**: Обработка текстовых сообщений

### 4. Утилиты (utils/)
- **logger.py**: Настройка логирования
- **helpers.py**: Вспомогательные функции

## Поток данных

```
Telegram API
    ↓
Application (main.py)
    ↓
Handlers (commands.py, messages.py)
    ↓
Utils (helpers.py)
    ↓
Response → Telegram API
```

## Технологический стек

- **Python 3.8+**: Основной язык программирования
- **python-telegram-bot**: Библиотека для работы с Telegram Bot API
- **python-dotenv**: Управление переменными окружения

## Расширяемость

### Добавление новых команд
1. Создайте функцию-обработчик в `bot/handlers/commands.py`
2. Зарегистрируйте обработчик в `bot/main.py`

### Добавление обработчиков сообщений
1. Создайте функцию-обработчик в `bot/handlers/messages.py`
2. Зарегистрируйте обработчик в `bot/main.py`
