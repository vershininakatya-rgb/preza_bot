### Первый запуск
1. Откройте Telegram
2. Найдите бота [@your_bot_username]
3. Нажмите "Запустить" или отправьте команду `/start`

## Основные команды

### /start
Начинает работу с ботом. Бот приветствует пользователя и предоставляет базовую информацию.

**Пример использования:**
```
/start
```

### /help
Показывает список всех доступных команд и их описание.

**Пример использования:**
```
/help
```
